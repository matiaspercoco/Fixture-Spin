# Fixture-Spin
# 🎡 La Agónica - Sorteador Oficial

**La Agónica** es una herramienta web de sorteos diseñada específicamente para el **Circuito Latinoamericano**. No es simplemente una ruleta; es un motor de suspenso diseñado para que la conformación de grupos y fixtures sea un evento cargado de adrenalina.

Gracias a su algoritmo de giro extendido y su frenado milimétrico, "La Agónica" asegura que cada resultado se viva con la máxima intensidad.

## 🚀 Características Principales

* **Motor de Incertidumbre:** Animación de 10 segundos con una curva de frenado personalizada (`cubic-bezier`) que hace que la ruleta parezca detenerse... para luego avanzar un último suspiro.
* **Integración con Excel:** Carga masiva de jugadores mediante archivos `.xlsx` respetando nombres y divisiones.
* **Gestor de Fixtures:** Organiza automáticamente los enfrentamientos "vs" a medida que salen los ganadores.
* **Exportación Profesional:** Genera y descarga el fixture final en un nuevo archivo Excel listo para compartir con la comunidad.
* **Estética Deportiva:** Interfaz optimizada en Azul Oxford y Dorado, ideal para streamings y presentaciones en vivo.

## 🛠️ Tecnologías

* **Core:** HTML5, CSS3 (Custom Animations), JavaScript (ES6+).
* **Librerías Externas:**
    * [ExcelJS](https://github.com/exceljs/exceljs) - Para la creación de reportes profesionales.
    * [SheetJS](https://github.com/SheetJS/sheetjs) - Para la lectura dinámica de datos.
    * [FileSaver.js](https://github.com/eligrey/FileSaver.js/) - Para la gestión de descargas en el cliente.

## 📦 Uso Rápido

1.  **Carga:** Subí tu archivo Excel con las columnas `Nombre` y `Division`.
2.  **Filtra:** Seleccioná la categoría que vas a sortear.
3.  **Gira:** Presioná el botón de acción y disfrutá del suspenso.
4.  **Descarga:** Una vez terminada la sesión, bajá el fixture oficial con un click.
